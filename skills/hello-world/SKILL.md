# Hello World

This is a demonstration skill. It shows how to structure a skill in this repository.

## Capabilities

1. **Say Hello**: Greets the user.
2. **Demonstrate Resources**: Shows how scripts, references, and assets are organized.

## Usage

When a user asks "say hello", this skill can be used to provide a friendly greeting.

## Bundled Resources

- \`scripts/example_script.cjs\`: A simple script that outputs a greeting.
- \`references/example_reference.md\`: A document explaining the "Hello World" concept.
- \`assets/example_asset.txt\`: A text file containing a "Hello World" template.
